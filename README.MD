# Mini CRUD - PHP + MySQL con Docker Compose

## Integrante
* Leonel Gonzalo Gonzalez (Luigi Gonzalez)

## Descripción del Proyecto
Este proyecto consiste en una aplicación web básica (CRUD) desarrollada en PHP que interactúa con una base de datos MySQL. Toda la infraestructura está contenerizada utilizando Docker y Docker Compose para garantizar un entorno de desarrollo limpio, aislado y de fácil despliegue.

El proyecto también sirvió como práctica para la gestión de versiones, aplicando un flujo de trabajo profesional con Git y GitHub que incluye el uso de ramas secundarias y fusiones (merges).

---

## Estructura de Desarrollo (Git & GitHub)
Para cumplir con los requerimientos de la asignación, el historial de versiones se gestionó de la siguiente manera:
* **Inicialización:** Repositorio inicializado localmente.
* **Historial de cambios:** Se realizaron al menos 3 commits descriptivos para registrar el avance de la estructura base y la documentación.
* **Gestión de Ramas:** Se creó una rama secundaria llamada `feature-login` para simular el desarrollo de un módulo de inicio de sesión, la cual posteriormente fue fusionada (*merged*) con la rama principal `main`.
